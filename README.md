## Documentation

### Installation

1. Clone this repo

```bash
$ git clone https://github.com/antoniofelices/js-test-level-fizz-buzz .
```

2. Install packages

```bash
$ npm install
```

3. Run Vite

```bash
$ npm run dev
```

4. Run tests

```bash
$ npm test:watch
```
